== Description ==

This plugin increases the efficiency of spam distinction of a comment. It is possible to extract URL from the past spam comment and add to a blacklist. Moreover, URL contained in a new spam comment can be added to a blacklist. 

= Some features: =

* The key of a blacklist and a moderation list is automatically rearranged in order of a name. 
* Each key is movable by drag and drop. 
* URL contained in the accumulated spam comment can be added to a blacklist.
* URL contained in a new spam comment can be added to a blacklist when marked spam. 
* The comment in which a few URL is contained can be marked as spam.
